|----------|-------------|
| Operator shell -> npm scripts | Local commands choose whether the API starts under Bun or Node. |
| Local workspace -> oracle SSH | Runtime availability checks cross into the remote host execution environment. |
| Package scripts -> production server | Script strings control the runtime binary used by the Express API. |

</threat_model>
