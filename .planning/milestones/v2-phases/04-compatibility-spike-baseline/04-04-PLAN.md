Capture the Node performance baseline (MEAS-01) with the low-variance methodology from RESEARCH.md and commit it to BASELINE.md so Phase 5 (MEAS-02/03) can run the identical method under Bun and produce a fair before/after comparison. A single human-verify checkpoint confirms the committed numbers are sane and the methodology is documented before the phase gate.
