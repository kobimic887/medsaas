## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Runtime process -> OS `/proc` | Measurement script reads host process RSS from Linux procfs. |
| Load generator -> Express API | Synthetic traffic crosses HTTP boundary into `/health` and `/health/db`. |
| Metrics report -> package scripts | Measured data controls whether Bun or Node is the final default runtime. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-03-01 | Tampering | `BUN-BEFORE-AFTER.md` gate values | mitigate | Report includes raw per-sample distributions, medians, deltas, and exact method so the gate can be audited. |
| T-05-03-02 | Denial of Service | Unsafe default runtime flip | mitigate | Gate compares Bun idle RSS against the fixed 118.9 MiB Node baseline and keeps Node defaults when Bun does not reduce idle RSS. |
| T-05-03-03 | Repudiation | Runtime measurement provenance | mitigate | Report records host, OS, runtime versions, commands, endpoint mix, and N=5 method. |
| T-05-03-04 | Information Disclosure | Env values in reports | mitigate | Capture script redacts MongoDB credentials and uses dummy Stripe/JWT values for test runs. |
| T-05-03-SC | Tampering | npm/pip/cargo installs | mitigate | Measurement creates no new dependencies and does not mutate lockfiles; it reuses existing npm-installed server modules and `spike/load-gen.mjs`. |
</threat_model>
