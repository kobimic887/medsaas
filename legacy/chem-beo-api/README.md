# Legacy Chem API (chem_beo)

This directory is a snapshot of the standalone **chem_beo** API (from [eitangenis/chem_beo](https://github.com/eitangenis/chem_beo) and [kobimic887/chem_beo](https://github.com/kobimic887/chem_beo)).

**Do not deploy this folder for new work.** The active application is:

- `server/` — current API (successor to chem_beo `index.js`)
- `client/` — React dashboard

Useful artifacts kept here:

- `diff_dock.sh` — also copied to `server/diff_dock.sh` for DiffDock script execution
- Historical scripts and utils for reference during migration
